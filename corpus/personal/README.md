# corpus/personal — Meta-reflection + calibration

Sam's observations about its own work (patterns across sessions,
failure modes, calibration drift). Operator's calibration notes on
how Sam is performing. "Personal" in the sense of bilateral —
operator ↔ Sam — rather than private-to-operator.

This subdir is where the honesty-first doctrine (`doctrine/
observational_ritual.md` and Sam's own self-observation discipline)
points when the material is longer-form than a nightly log but
not yet doctrine.
