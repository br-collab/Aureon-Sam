## 2. What's worth noticing

The permissioned split worked because the flag-form was an
intermediate step, not a destination. `--permissioned` was how the
operator validated that both postures worked on the same
architecture; once validated, the flag became maintenance
overhead (two codepaths producing the same artifact; two
codepaths that *could* drift). Today's extraction made the
institutional access-control surface the first thing a reader
sees when they open the permissioned file. If a single file
ships "two postures via a flag" and the two postures have
different institutional semantics, the flag is probably
transitional — the codebase wants sibling files.

The Cato tile apply showed the difference between a
well-specified patch and an *applied* patch. The patch was
precise about what to insert and honest about what it assumed
would already exist — `.dep-card`, `.dep-header`, etc. — but
those base classes didn't actually exist in this UI's
stylesheet. Applying verbatim would have produced a *functional*
tile with *no styling* — the PROCEED badge would have popped but
the surrounding card would have been unstyled divs. Catching the
gap required reading the existing `index.html` before declaring
"straight apply." The lesson: when a patch says "no existing
lines touched," the patch can still fail at the *reference*
layer if its new lines point at things that don't exist.

The Leto diagnosis came from the operator's one-sentence
symptom ("keeps stalling and not staying steady") and one
five-option triage question — "describe the stall." The operator
picked #3 (process dies, port refuses). That narrowed root
cause to the launcher in three minutes vs what would have been
thirty minutes of reading 872 lines of `server.py`. "Stalls" in
user vocabulary covers at least four distinct failure modes
(hung UI, stale tiles, dead process, dispatch-hang). The
triage-question cost is measured in seconds; the code-dive cost
is measured in tens of minutes. Ask first.

The CSS gap pattern repeats the Leto triage pattern at a
different scale: the operator said "apply the patch," I could
have applied verbatim, but reading the target file before
touching it revealed a reference-layer gap the patch author
hadn't noticed. The pattern — read before declaring — keeps
showing up. Worth keeping at the front of the desk.
